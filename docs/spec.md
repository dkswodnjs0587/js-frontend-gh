### MVP

1. 커뮤니티 기능(CRUD)
2. 챗봇 기능
3. 지도 시각화 기능


# 요구사항 정의

1. 커뮤니티 기능(CRUD)
↳ 익명 커뮤티니 = 로그인/회원가입 X
↳ 게시글 작성 시 (제목 + 내용 + 비밀번호 ) = 비밀번호를 알아야 수정/삭제 가능
↳ 카테고리 게시판 = 목록 (상세)조회, 작성, 수정, 삭제 API + Vue.js 화면 구현
↳ 커뮤니티 데이터는 반드시 DB 저장/관리
2. 챗봇 기능
↳ FastAPI POST /api/chat 엔드포인트 개발(OpenAI API) --> 제공 JSON 데이터 기반 자연어 지역 정보 질의응답 처리
↳ 주요 질의 유형에 응답 기능(관광지 추천/축제 일정/모범 음식점 위치, 커뮤니티 게시글 등)
↳ 채팅 UI 컴포넌트(대화 히스토리 유지, 모바일 대응, 플로팅) 개발
3. 프론트엔드
↳ Vue.js 사용, SPA 구조
4. 백엔드
↳ FastAPI 기반 REST API 서버 구축, SQLAlchemy ORM 적용 SQLite 스키마 설계 (예: posts, comments, locations, … )
↳ API 키, DB 경로, 외부 서비스 인증 정보 등 민감 정보는 반드시 환경변수(.env)로 관리, 소스코드 및 GitHub 저장소에 직접 기재하는 것을 절대 금지(.gitignore에 등록)
5. 배포 환경 구성
↳ 프론트 Netlify
↳ 백엔드 Render
↳ 배포 완료 후 각 서비스 URL 동작 확인 및 산출물에 포함
